# Minimum Requirements Checklist

These are required for any app to be listed on the HubSpot marketplace.

## Functional Integrity

- [ ] App uses a single HubSpot app ID
- [ ] App authenticates via OAuth (not API keys for marketplace apps)
- [ ] App functions independently without requiring other third-party apps
- [ ] App performs the functionality described in its listing

## Active Usage

- [ ] 3+ active installs from unaffiliated customer accounts
- [ ] Verified through OAuth-authenticated API requests or signed webhook requests
- [ ] Activity within the past 30 days

## Scope Discipline

- [ ] Only requests OAuth scopes the app actually uses
- [ ] No unused scopes in required scope list
- [ ] Context-specific scopes moved to optional/conditional where appropriate

## Eligibility

- [ ] App is not available for restricted industries
- [ ] App does not violate HubSpot's Acceptable Use Policy

## Review Process

- Initial review: 10 business days
- Complete process: up to 60 days from feedback
- HubSpot Ecosystem Quality team performs manual review and functional testing
