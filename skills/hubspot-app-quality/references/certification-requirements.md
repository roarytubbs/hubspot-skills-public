# Certification Requirements

Certification is a higher quality tier for apps already listed on the marketplace. Certified apps receive a badge, search filter visibility, curated collection eligibility, and priority developer support.

Certification is valid for 2 years (rolling). Recertification notifications are sent 90 days before expiration.

## Prerequisites

- [ ] App meets all minimum and listing requirements
- [ ] App has been listed for at least 6 months
- [ ] Good standing: no violations, no rejection in last 6 months, no unresolved support escalations

## 1. Security and Privacy

- [ ] Single app ID with OAuth authentication
- [ ] Associated with a verified domain
- [ ] All requested scopes are actively used — remove unused ones
- [ ] Security assessment of public assets passes review
- [ ] No unauthorized API usage or undocumented data access
- [ ] Secrets kept off client-side code
- [ ] Request signatures validated for external backends

## 2. Reliability and Performance

- [ ] 95%+ API success rate across all activities
- [ ] Respects rate limits (100 requests per 10 seconds per account)
- [ ] Proper token management: refresh tokens before expiry, cache appropriately
- [ ] Retry/backoff implemented for transient errors
- [ ] Reasonable activity volume from unaffiliated customers
- [ ] No excessive polling or unnecessary API calls

## 3. Usability and Accessibility

- [ ] Clear marketplace listing accurately describes current functionality
- [ ] App does not cause confusion or frustrate core HubSpot workflows
- [ ] Installation and OAuth flow works cleanly for new accounts
- [ ] Reconnect/reauthorization flow handles token expiry gracefully
- [ ] Free trial or freemium model available (recommended)

## 4. Value Indicators

- [ ] 60+ active installs minimum (vs. 3 for basic listing)
- [ ] Positive retention metrics (installs growing or stable, not declining)
- [ ] Marketplace reviews demonstrate customer value
- [ ] Responsive to customer reviews, especially negative ones

## 5. Documentation Quality

- [ ] All documentation is live and publicly accessible (no paywalls or login gates)
- [ ] GDPR-compliant (no unauthorized tracking on doc pages)
- [ ] Clear descriptions of: what the app does, installation, configuration, usage, uninstallation
- [ ] Screenshots are current and consistent with HubSpot design system
- [ ] Video walkthroughs recommended and regularly updated

## 6. Customer Engagement

- [ ] Responds to marketplace reviews in a timely manner
- [ ] Negative reviews are addressed constructively
- [ ] Support inquiries resolved within reasonable timeframes

## 7. Compliance and Standing

- [ ] No policy violations in the past 6 months
- [ ] No listing rejections in the past 6 months
- [ ] No unresolved HubSpot support escalations
- [ ] Passes periodic compliance checks during certification window

## Certification Benefits

- "HubSpot Certified App" badge on marketplace listing
- Inclusion in "HubSpot Certified App" search filter
- Eligibility for curated marketplace collections
- Favorable consideration for partnership and amplification initiatives
- Priority developer support access

## Timeline

- Initial review: 10 business days
- Full certification process: up to 60 days
- Certification duration: 2 years (rolling)
- Renewal notification: 90 days before expiration
